# Memory-Vision
## The instant classic, a Memory game! This Memory game is limited by a timer and by user moves, that make the game a bit challenging.

## Link to Web Application
https://memory-vision.netlify.app

## Build With
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>SASS</li>
  <li>JAVASCRIPT</li>
  <li>GSAP</li>
</ul>

## Features
<ul>
  <li>Accurate Game Timer</li>
  <li>Beautiful GSAP animations</li>
  <li>Simple design</li>
  <li>Limited User Moves</li>
  <li>Properly challenging</li>
</ul>
